PRISM-Δ

Projection-based Relevance-Informed Steering Method

Training-free attention steering via differential cross-covariance decomposition.
Compatible with FlashAttention. Negligible memory overhead.

arXiv Code 🤗 Dataset

How It Works

Differential Cross-Covariance

SVD of ΩΔ = Ω+ − Ω− extracts directions that distinguish relevant from irrelevant conditions, automatically eliminating shared variance.

Softplus Head Weighting

Each attention head receives a continuous importance weight based on its discriminability score, replacing binary hard thresholds with smooth gradation.

Dual-Channel Steering

Optionally steers both Key (routing) and Value (content) channels simultaneously, enabling fine-grained control over information flow.

Quick Start

terminal 
$ pip install -r requirements.txt

# Build PRISM-K projection
$ python src/custom_builders/synthetic_qa_builder.py \
    --model Qwen/Qwen3-4B-Base \
    --data data/synthetic/pair_qa_new.jsonl \
    --output_dir ./projections --diff-only

# Run evaluation with PRISM-K steering
$ python benchmarks/eval_bias_gen.py \
    --model Qwen/Qwen3-4B-Base \
    --data_path data/biasbios/biasbios.json \
    --output_dir ./results/prism-k \
    --prism-k --prism-k-proj ./projections/*_diff_proj.pt \
    --prism-k-gain 0.40 --layers all
